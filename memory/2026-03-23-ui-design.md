## Design System Features

### Visual Style
- **Dark-first design** with #0a0a0a background
- **Coral/orange accents** (#ff6b4a) for CTAs and highlights
- **Glassmorphism effects** for overlays and cards
- **Geometric patterns** for visual interest
- **Bold typography** with Space Grotesk for headings

### Navigation
- **Horizontal top bar** (fixed position)
- Logo on left, nav links in center, user menu on right
- Notification bell with badge
- Smooth hover effects with underline animation
- Mobile: Bottom navigation bar

### Components
- Buttons (4 variants: primary, secondary, ghost, icon)
- Cards (standard + glass effect)
- Status badges (color-coded)
- Modal dialogs (with backdrop blur)
- Toast notifications (4 types)
- Form inputs (text, select, checkbox)
- Loading states (spinner + skeleton)

### Animations
- Framer Motion for page transitions
- Hover effects (lift + glow)
- Smooth transitions (200-300ms)
- Stagger animations for lists

## Key Design Decisions

1. **Dark theme by default** - Professional, modern look
2. **Horizontal navigation** - More space for content
3. **Coral accent color** - Warm, energetic, stands out
4. **Glassmorphism** - Modern, depth, visual hierarchy
5. **Geometric patterns** - Visual interest without distraction
6. **Bold typography** - Clear hierarchy, easy to scan
