### COLMAP Usage Clarification

User asked about COLMAP dependency for Gaussian Splatting:

**Answer:** Yes, Gaussian Splatting needs COLMAP output, but COLMAP itself is only used in Phase 2.

**Flow:**
1. **Phase 2 (Video Processing)** - COLMAP runs here:
   - Task 6: Camera Pose Estimation
   - Extracts SIFT features, matches them, performs sparse reconstruction
   - Outputs: cameras.bin, images.bin, points3D.bin

2. **Phase 3 (Gaussian Splatting)** - Uses COLMAP output:
   - Task 9: Loads COLMAP sparse points
   - Initializes Gaussians from these points
   - Trains the model

**For 3D file imports (Phase 4):** COLMAP not needed - files already have geometry.

**Fallback:** System has fallback if COLMAP unavailable (uses random initialization, lower quality).

**Alternative:** Task 15.5 (Camera Metadata Import) allows importing camera data from other sources (CSV, XML, etc.) to bypass COLMAP.

Created: `docs/COLMAP_USAGE_EXPLAINED.md` with full explanation.
