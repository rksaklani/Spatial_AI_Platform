Geometric Background

This is a test of the geometric background pattern inspired by Nira.app. The pattern uses CSS gradients to create a subtle geometric texture that adds visual interest without overwhelming the content.

Test Button
Pattern Details:
• Dark gradient background (#0a0a0a to #1a1a1a)
• Coral accent color (#ff6b4a) at 5% opacity
• 80px × 140px repeating pattern
• Pure CSS - no images required
• Performance: Excellent
